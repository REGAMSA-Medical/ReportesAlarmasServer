# 📊 REGAMSA OrdersTrackingApp (FastAPI)
Servidor de la app de automatización para la generación de reportes de ventas y gestión de pedidos y ordenes.

## 🚀 Características
- 📈 **Base de datos local con conexión remota:** Se usa AWS IoT Core para acceder a la DB desde cualquier lugar.
- 📝 **Gestión de Pedidos:** Control de flujo de órdenes a nivel de datos.
- 📄 **Reportes Automáticos:** Importación/Exportación a PDF/Excel usando Pandas para la gestión de los datos.

## 🛠️ Tecnologías
- **Backend:** FastAPI & Python.
- **Database:** PostgreSQL
